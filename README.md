# 📊Data Science Internship 

End-to-end machine learning projects covering classification, regression, NLP and data analysis — each deployed as an interactive Streamlit web application.

## 📹 Demo

## 📂 Projects

### 🌸 Task 1 — Iris Flower Classification

**Objective -** Classify iris flowers into three species (*Setosa*, *Versicolor*, *Virginica*) based on sepal and petal measurements.

| Feature | Details |
|---|---|
| Dataset | Sepal Length, Sepal Width, Petal Length, Petal Width |
| Model | Random Forest Classifier |
| App | Input flower measurements → predict species |

**Highlights -** Data preprocessing → EDA → Model training → Streamlit UI

### 📉 Task 2 — Unemployment Analysis

**Objective -** Analyze unemployment trends across Indian regions, with a focus on the impact of the COVID-19 pandemic.

| Feature | Details |
|---|---|
| Dataset | Date, Region, Unemployment Rate, Labour Participation Rate |
| Analysis | Time series, regional comparison, COVID-era trends |
| Visuals | Line charts, bar charts, heatmaps |

**Highlights -** Interactive Streamlit dashboard for exploring regional unemployment data.

### 🚗 Task 3 — Car Price Prediction

**Objective -** Predict the selling price of used cars based on vehicle attributes.

| Feature | Details |
|---|---|
| Dataset | Present Price, KMs Driven, Fuel Type, Transmission, Owner, Car Age |
| Model | Random Forest Regressor / XGBoost |
| App | Input car details → get predicted selling price |

**Highlights -** Feature engineering + categorical encoding + ensemble learning for strong predictive performance.

### 📧 Task 4 — Email Spam Detection

**Objective -** Classify emails/SMS messages as **Spam** or **Ham** using NLP.

| Feature | Details |
|---|---|
| Dataset | Thousands of labeled SMS/email messages |
| Techniques | Text cleaning, tokenization, stopword removal, TF-IDF |
| Model | Multinomial Naive Bayes |
| Accuracy | ~97% |

**Highlights -** Streamlit app where users paste any email text for instant spam detection.

### 📈 Task 5 — Sales Prediction

**Objective -** Predict product sales based on advertising budgets across TV, Radio, and Newspaper channels.

| Feature | Details |
|---|---|
| Dataset | TV, Radio, Newspaper ad spend → Sales |
| Model | Random Forest Regressor |
| App | Input ad budgets → estimate expected sales |

**Highlights -** Advertising impact analysis + interactive visualizations for business decision support.
